# **App Name**: ImmersioVerse

## Core Features:

- 3D Project Showcase: Display developer projects using interactive 3D elements and holographic cards, built with React Three Fiber.
- Tech Stack Visualizer: Visually represent the tech stack (React, JavaScript, Firebase, MERN) with rotating 3D icons and glowing effects.
- Digital Art Gallery: Showcase drawings, photo edits, and creative works in a dynamic gallery with floating frames and physics-based animations.
- Interactive Depth Scrolling: Implement depth-based scrolling effects to create a sense of immersion as users navigate through the portfolio.
- Generative 3D tool Integration: Allow the user to specify certain visual qualities they want and generate 3D environments and components in a style suited to the user, incorporating generated results throughout the site.
- Contact Portal: Provide a contact portal for recruiters and clients that is easy to use and clearly visible.

## Style Guidelines:

- Primary color: Dark, desaturated grayish blue (#4682A9) to evoke a sense of professionalism and innovation.
- Background color: Near-black (#121212) to enhance the immersive 3D environment.
- Accent color: Pale gold (#D4A27A) for highlights, interactive elements, and calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-forward and modern aesthetic, and 'Inter' (sans-serif) for body text.
- Use futuristic, minimalist icons with subtle glowing effects to represent tech skills and project categories.
- Implement a split-screen design with the 'Developer Side' and 'Artist Side' seamlessly connected through interactive transitions.
- Incorporate smooth physics-based animations and subtle motion that reacts to cursor movement and scrolling.
